# Countdown to ABC

Simple static webpage that displays a large red **1234** centered on the page and counts down **10 seconds** before revealing a button linking to **https://abc.com**.

## Files
- `index.html` — page markup
- `styles.css` — styling
- `script.js` — countdown logic

## Usage
1. Clone or copy the files into a repository.
2. Serve the folder (open `index.html` in a browser or use a static host like GitHub Pages).
3. The page shows **1234** in the center; after 10 seconds the button appears.

## Notes
- The link opens in a new tab.
- You can change the countdown length by editing `START_SECONDS` in `script.js`.
